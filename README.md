# Agri-Blog
Agri Blog is a full-stack web application that merges an e-commerce platform with an agricultural blog. Built using HTML, CSS, JavaScript, Node.js, and EJS, it enables users to browse and purchase agri-products online while reading expert-driven blog content related to modern farming practices, innovations, and sustainability. The platform aims to empower farmers, agri-enthusiasts, and consumers by providing both agricultural knowledge and a convenient shopping experience in a single website.
